# Create .test. file

**Create .test. file** is a VS Code extension that helps you create unit test files effortlessly. Simply right-click on any `.js`, `.ts`, `.jsx` or `.tsx` file in the Explorer, and select "Create .test. file" from the context menu. A matching `.test.` file will be generated in the same directory, ready for you to write those all-important tests.
